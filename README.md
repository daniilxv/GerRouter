# GeoRouter 🗺️

**GeoRouter** — это веб-приложение на базе Django для планирования путешествий. Оно позволяет пользователям создавать маршруты, разбивать их по дням и отмечать ключевые точки (waypoint) на карте.

## ✨ Основные возможности

- **Управление пользователями**: Регистрация, авторизация и выход из системы.
- **Планирование поездок**: 
    - Создание нескольких именованных путешествий.
    - Разделение каждого путешествия на отдельные дни.
    - Добавление географических точек (широта и долгота) для каждого дня.
- **Интерактивная карта**: Визуализация маршрутов и удобное управление точками (через интерфейс карты).
- **Сохранение данных**: Возможность сохранять и загружать маршруты с помощью JSON API.
- **Список поездок**: Просмотр всех своих созданных маршрутов и возможность их удаления.

## 🛠 Технологический стек

- **Backend**: Python 3.x, Django
- **Frontend**: HTML, CSS, JavaScript (интеграция с картами)
- **База данных**: SQLite (по умолчанию)

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone <url-репозитория>
cd GeoRouter
```

### 2. Создание виртуального окружения
```bash
python -m venv venv
source venv/bin/activate  # Для Linux/macOS
# или
venv\Scripts\activate     # Для Windows
```

### 3. Установка зависимостей
```bash
pip install django
```

### 4. Настройка базы данных
```bash
python manage.py migrate
```

### 5. Создание суперпользователя (опционально)
```bash
python manage.py createsuperuser
```

### 6. Запуск сервера
```bash
python manage.py runserver
```
После запуска приложение будет доступно по адресу `http://127.0.0.1:8000/`.

## 📂 Структура проекта

- `core/` — основные настройки проекта (`settings.py`, `urls.py`).
- `router/` — основное приложение для планирования маршрутов:
    - `models.py` — модели данных (`Trip`, `Day`, `Waypoint`).
    - `views.py` — логика обработки запросов и API для работы с картой.
    - `urls.py` — маршрутизация приложения.
    - `templates/` — HTML-шаблоны страниц (карта, список поездок, авторизация).
- `manage.py` — стандартный скрипт управления Django.
